## USAGE

```
echo "
src-git base https://git.openwrt.org/openwrt/openwrt.git
src-git packages https://git.openwrt.org/feed/packages.git
src-git extra https://github.com/khadas/khadas-openwrt-feed-extra
" > feeds.conf
```
